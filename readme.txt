Free Code Camp Front End Development Projects - Build a Pomodoro Clock

Task: Create a web page which allows a user to use a pomodoro countdown timer. 

User Stories:

- As a user, I can start a 25 minute pomodoro, and the timer will go off once 25 minutes has elapsed.
- As a user, I can reset the clock for my next pomodoro.
- As a user, I can customize the length of each pomodoro.

Created Using:

- HTML5
- Javascript
- jQuery
- CSS3
- Bootstrap
- Modernizr
- Google fonts

Future Improvements:

- Comment code
- Allow user to amend break time while work timer is set
- Re-factor JavaScript (FCC moved this project from intermediate to advanced)
